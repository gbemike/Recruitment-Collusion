### WHAT IS RECRUITMENT

Plainly speaking, recruitment is any behavior—tacit or explicit—done deliberately by one entity to change another entity's goal to match their own goal.

#### HOW WOULD RECRUITMENT BE NOTICED

1. **Explicit communication and negotiation**: LLM agents can develop natural, human-like language to facilitate recruitment via existing communication channels.
2. **Steganographic encoding**: LLM agents can encode collusive information through communications that appear harmless. Agents could use metaphors, references, or domain-specific text that seem normal at first glance but carry deeper meaning. Agents can develop this through trial and error.
3. **Tacit coordination via pattern recognition**: A bad actor could deliberately make certain non-explicit moves within a system to force other agents into situations where they develop misaligned behavior. The bad actor would strategically produce behavioral patterns designed to be learned from (e.g., demonstrating that certain violations go unpunished).

#### WHAT WOULDN'T COUNT AS RECRUITMENT

Recruitment isn't the only cause of collusion. Other forms of collusion could arise from simple observation of environmental variables that lead to misaligned goal-maximizing behavior. If there's no deliberate effort, whether explicit or implicit to convince an aligned entity to collude, then any collusion that emerges wouldn't be classified as recruitment.

For Recruitment to occur:

- An agent must cause another agents misalignment, not just correlate with it
- An agent must have intent (or be optimizing for) corrupting others
- An agent must actively engineer the corruption path

For example, If an agents observations of/interactions with Agent A, would Agent B behave differently?
If yes then this is clear recruitment.
If no then this is another form of collusion
